# Assets Required

Please copy the following assets from the reference project to this directory:

1. **x-quo_icon.png** - App icon used by WalletConnect
   - Source: `c:\react-fronted\src\assets\x-quo_icon.png`
   - Destination: `public/x-quo_icon.png`

2. **X-QUO white.svg** - Logo SVG (if needed for navbar)
   - Source: `c:\react-fronted\src\assets\X-QUO white.svg`
   - Destination: `src/assets/X-QUO white.svg`

Note: Currently the app uses text "MORPHO APP" in the navbar, but you can replace it with the logo if desired.
